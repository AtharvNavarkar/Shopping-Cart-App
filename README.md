Shopping Cart App

This is a simple Shopping Cart web application made using JavaScript.
It allows users to view products, add them to cart, remove items, and see the total price before checkout.

Features
  Displays list of products
  Add products to cart
  Remove products from cart
  Shows total price of items in cart
  Shows message when cart is empty
  Simple checkout functionality

Technologies Used
  HTML – Page structure
  CSS – Styling
  JavaScript – Logic and DOM manipulation
  
How the App Works
  The app shows a list of available products.  
  User clicks Add to cart button to add a product.
  Selected product is added to the cart list.
  Total price is updated automatically.
  User can click Remove button to remove a product from cart.
  If cart is empty, message "Your cart is empty" is shown.
  When user clicks Checkout:
  If cart has items → success message is shown
  If cart is empty → warning message is shown
